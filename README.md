# OpenFable

> **Fable is gone. Long live Fable.**

**Verified output. Open to everyone.**

The autonomous output quality that made Fable famous - now open, accessible, and built for everyone.

## What We're Building

An open-source verified output engine. The kind of quality that used to be locked behind enterprise contracts and corporate subscriptions - now available to anyone.

- **Silent verification** - output is refined before you see it, no visible machinery
- **Open source** - MIT, self-hostable, transparent
- **Uncensored** - no corporate filters
- **Affordable** - or free if you self-host

## Status

Early stage. Building in the open. Not shipped yet.

Built on the work of [KingLabsA](https://github.com/KingLabsA) and the FableForge community.
